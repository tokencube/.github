# Tokencube

We build **[Zero](https://zero.ski)** — a skill-native agent runtime.

> *Agents need Skills to ski.*

Every capability — chat, search, billing, email, UI — is a Skill. A small kernel binds them together; everything else is a Skill you can read, compose, or replace.

## Open

- **[zeroski](https://github.com/tokencube/zeroski)** — official CLI for the Zero runtime.
  `cargo install zeroski`

## Closed (for now)

Zero's kernel, the Skill marketplace, and domain Skills (MRO, DeepEDU) live in private repos and will open as they harden. Follow `zero.ski` for release notes.
